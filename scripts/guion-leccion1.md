## Desarrollo (3 min)

### ¿Qué es una página web?

Una página web es un documento que se muestra en tu navegador, ya sea Chrome, Firefox, Safari o Edge.

Cuando escribes una dirección como `google.com` o `youtube.com`, tu navegador descarga ese documento y lo muestra en pantalla.

Pero ese documento no es magia. Está escrito en un lenguaje especial que los navegadores entienden.

<br>

### ¿Qué es HTML?

HTML significa HyperText Markup Language, o en español: Lenguaje de Marcado de Hipertexto.

HTML es el lenguaje que define el **contenido** de una página web.

Con HTML le dices al navegador qué mostrar:

- Un título.
- Un párrafo de texto.
- Una imagen.
- Un enlace a otra página.

Piensa en HTML como el esqueleto de una página web. Define la estructura, lo que hay y en qué orden aparece.

<br>

### ¿Qué es CSS?

CSS significa Cascading Style Sheets, u Hojas de Estilo en Cascada.

CSS es el lenguaje que define el **aspecto visual** de una página web.

Con CSS le dices al navegador cómo mostrar el contenido:

- De qué color es el texto.
- Qué tan grande es un título.
- Qué fondo tiene la página.
- Cómo están distribuidos los elementos.

Piensa en CSS como la ropa y decoración de ese esqueleto. Le da estilo, color y personalidad.

<br>

### HTML y CSS trabajan juntos

Un sitio web = Contenido (HTML) + Estilo (CSS).

Sin HTML no habría nada que mostrar.
Sin CSS todo se vería aburrido y sin formato.

Juntos crean las páginas atractivas y organizadas que ves todos los días en internet.

Y lo más importante: **cualquier persona puede aprender a crearlas**, incluso tú, desde hoy.
